# AWS & Terraform Internship Assignment 

This repository contains:

- Overview of important AWS services
- Introduction to Terraform (IaC tool)
- Simple Terraform example to create S3 bucket + EC2 instance on AWS
- Screenshots and explanations

## Files
- **AWS assignment.docx** → Detailed explanation & notes
- **provider.tf** → AWS provider configuration
- **main.tf** → Terraform resources (S3 + EC2)
- **image1.jpeg** → AWS EC2 console screenshot after deployment

## How to use the Terraform code
1. Install Terraform
2. Configure AWS CLI credentials
3. Run: `terraform init`, `terraform plan`, `terraform apply`
